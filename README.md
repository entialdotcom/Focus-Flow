# FocusMode

A focus and productivity app with curated audio for deep work, relaxation, meditation, and motivation.

## What It Does

FocusMode helps you get into the zone with curated YouTube audio streams. Choose a mode that fits your goal - Focus for deep work, Relax for unwinding, Sleep for rest, Meditate for mindfulness, or Motivation for inspiration. Track your progress with listening streaks and unlock badges as you build consistent focus habits.

## Use Cases

- **Remote workers**: Block out distractions with focus-enhancing music
- **Students**: Create the perfect study environment for long sessions
- **Writers & creatives**: Get into flow state with ambient soundscapes
- **Meditators**: Access guided meditations across multiple themes
- **Anyone stressed**: Unwind with relaxation tracks after a long day
- **Morning routines**: Start your day with motivational content
- **Sleep improvement**: Fall asleep with calming ambient audio

## Modes

| Mode | Purpose | Content |
|------|---------|---------|
| **Focus** | Deep work & concentration | Lo-fi beats, ambient music, productivity playlists |
| **Relax** | Unwinding & stress relief | Chill music, nature sounds, calming instrumentals |
| **Sleep** | Rest & recovery | Sleep music, ASMR, ambient soundscapes |
| **Meditate** | Mindfulness & reflection | Guided meditations for gratitude, peace, abundance |
| **Motivation** | Inspiration & energy | Motivational speeches on discipline, success, mindset |

## Features

- Five distinct audio modes for different mental states
- Curated YouTube streams - no searching required
- Ambient sound mixer to layer sounds (rain, fire, wind, etc.)
- Listening streak tracking for building habits
- Badge system with achievements for hours listened and streaks maintained
- Favorite tracks to quickly access your best finds
- Auto theme switching (light during day, dark at night)
- Timer modes: infinite, countdown, or intervals
- Session logging to track your focus time
- Offline profile persistence

## Badges

Earn badges as you build your focus practice:

**Listening Hours**
- First Hour (1h)
- 10 Hours In (10h)
- Rising Star (20h)
- Focus Master (50h)
- Century Club (100h)

**Streaks**
- 3-Day Streak
- Week Warrior (7 days)
- Monthly Master (30 days)
- Streak Legend (50 days)
- Diamond Streak (75 days)
- Centurion (100 days)

## How to Use

1. **Pick a mode**: Choose Focus, Relax, Sleep, Meditate, or Motivation
2. **Select a category**: Browse subcategories within each mode
3. **Start listening**: Audio begins playing automatically
4. **Add ambient sounds**: Open the mixer to layer in rain, fire, or other sounds
5. **Favorite tracks**: Save tracks you love for quick access later
6. **Track progress**: View your stats, streaks, and badges in your profile
7. **Set a timer**: Use infinite mode or set a countdown for timed sessions

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm test
```

## Tech Stack

- React 19
- TypeScript
- Vite
- YouTube IFrame API for audio playback
- Vitest for testing

## Audio Sources

FocusMode uses curated YouTube videos as audio sources. All content is streamed directly from YouTube - no audio is stored or redistributed.

## Privacy

- Profile data stored locally in your browser
- No accounts required
- No tracking or analytics
- Your listening history stays on your device
